# 全渠道对账与财务凭证

## 技能概述

**技能名称**: reconciliation

**核心能力**: 统一处理全渠道账单对账，自动生成财务凭证，对接金蝶/用友等财务ERP。

## 能力描述

### 核心功能

- **全渠道账单汇集**: 统一接入各平台（天猫、京东、抖音、微信支付、支付宝等）的账单
- **自动化对账**: 自动核对订单流水与平台账单，发现差异自动标记
- **财务凭证生成**: 根据对账结果生成标准财务凭证

### 对账场景

**日常对账**:
```
天猫账单: 订单金额 - 平台扣点 - 推广费 = 实收
OMS订单: 商品销售额 + 运费 - 优惠 = 订单收入

对账差异: 天猫实收 vs OMS订单收入
```

**业财一体化**:
```
OMS订单 → 收入确认 → 分润计算 → 凭证生成 → 推送ERP

凭证分录:
  借: 应收账款-平台  100
  贷: 收入-商品      85
  贷: 增值税         15
```

## 调用示例

### 自然语言解析场景

**用户说**: "生成2024年1月天猫渠道的对账报表"

**AI解析后调用**:
```
reconciliation.generate_report(
  channel="tmall",
  period="2024-01",
  report_type="reconciliation"
)
```

**返回**: 对账报表（含订单数、销售额、平台扣点、退款金额、净收入），差异明细。

---

**适用场景**:
- 财务人员日常对账
- 运营分析各平台经营数据
- 审计人员核查账务一致性
