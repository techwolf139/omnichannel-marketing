# 逆向物流与残次鉴定

## 技能概述

**技能名称**: returns_logistics

**核心能力**: 处理退货物流的回收、质检、残次鉴定和库存再流转。

## 能力描述

### 核心功能

- **逆向物流追踪**: 全程追踪退货包裹轨迹，确认收货
- **残次鉴定**: 指导门店/仓库进行退货物况鉴定
- **库存智能再流转**: 根据鉴定结果自动处理库存

### 逆向物流流程

```
用户寄出 → 物流运输 → 退回仓库/门店 → 
质检鉴定 → 库存处理 → 退款确认
```

### 残次鉴定标准

| 等级 | 物况 | 处理方式 |
|------|------|----------|
| A级 | 全新未拆封/吊牌还在 | 就地转可售 |
| B级 | 已拆封但无损 | 折价转可售 |
| C级 | 有瑕疵但可用 | 转入特卖渠道 |
| D级 | 残次/损坏 | 维修或报废 |

### 就地再销售

**场景**: 门店收到线上退货，鉴定为B级品

**处理**: 系统自动将这件商品标记为"门店可售"，即刻上架门店库存

**优势**: 无需寄回总仓，极大降低物流成本和库存折旧

## 调用示例

### 自然语言解析场景

**用户说**: "帮我查一下退货单R789456的质检结果"

**AI解析后调用**:
```
returns_logistics.get_quality_result(return_id="R789456")
```

**返回**: 质检等级、鉴定时间、库存去向（就地销售/退回大仓/报废）。

---

**适用场景**:
- 仓库/门店员工执行质检任务
- 客服查询退货鉴定进度
- 运营分析退货物况分布
