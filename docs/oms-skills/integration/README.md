├─────────────────────────────────────────────────────────────┤
│  Phase2 订单路由                                            │

│  Phase3 库存管理                                            │

├─────────────────────────────────────────────────────────────┤
│  Phase4 售后                                                │
│  ├── returns_crosschannel (跨渠道退换)                     │

├─────────────────────────────────────────────────────────────┤
│  Phase5 财务                                                │
│  ├── profit_sharing (分润)                                 │
│  └── reconciliation (对账)                                 │

          ▲                                    ▲
          │ 营销推广系统                       │ 商品销售系统

## 核心集成点

### 订单获取

| 平台 | 订单接入 | 状态同步 |
|------|----------|----------|
| 天猫/淘宝 | API拉取 + Webhook | 实时 |
| 京东 | API拉取 | T+1 |
| 拼多多 | API拉取 | 实时 |
| 抖音电商 | Webhook | 实时 |

### 库存同步

| 平台 | 库存回写 | 限流 |
|------|----------|------|
| 天猫 | 预占库存API | 1000/分钟 |
| 京东 | 库存更新API | 500/分钟 |
| 拼多多 | 库存更新API | 300/分钟 |
| 抖音 | 库存更新API | 1000/分钟 |

### 营销数据

| 平台 | 数据类型 | 归因权重 |
|------|----------|----------|
| 小红书 | 种草数据 | 0.15 |
| 抖音 | 直播数据 | 0.15 |
| 微信 | 私域数据 | 0.20 |
| 广告投放 | 点击数据 | 0.25 |
