## 核心架构

```
┌──────────────────────────────────────────────────────────────┐
│                    跨平台统一系统架构                         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  应用层 ─── 精准营销 / 智能客服 / 导购赋能 / 数据分析        │
│                                                              │
│  中台层 ─── 订单中台 / 流量中台 / 推荐中台 / One-ID Hub     │
│                                                              │
│  数据层 ─── 数据湖 / 特征中心 / 模型中心                      │
│                                                              │
│  接入层 ─── 销售平台(天猫/京东/拼多多)                       │
│              营销平台(小红书/抖音/微信)                       │
│              自有平台(品牌小程序/门店/企微)                    │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 核心集成点

### 订单获取

| 平台 | 订单接入 | 状态同步 |
|------|----------|----------|
| 天猫/淘宝 | API拉取 + Webhook | 实时 |
| 京东 | API拉取 | T+1 |
| 拼多多 | API拉取 | 实时 |
| 抖音电商 | Webhook | 实时 |

### 库存同步

| 平台 | 库存回写 | 限流 |
|------|----------|------|
| 天猫 | 预占库存API | 1000/分钟 |
| 京东 | 库存更新API | 500/分钟 |
| 拼多多 | 库存更新API | 300/分钟 |
| 抖音 | 库存更新API | 1000/分钟 |

### 营销数据

| 平台 | 数据类型 | 归因权重 |
|------|----------|----------|
| 小红书 | 种草数据 | 0.15 |
| 抖音 | 直播数据 | 0.15 |
| 微信 | 私域数据 | 0.20 |
| 广告投放 | 点击数据 | 0.25 |
