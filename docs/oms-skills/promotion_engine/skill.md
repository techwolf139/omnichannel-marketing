# 跨场景促销规则引擎

## 技能概述

**技能名称**: promotion_engine

**核心能力**: 配置和执行打通线上线下的复杂营销玩法，支持积分、优惠券、赠品等多种促销形式。

## 能力描述

### 核心功能

- **促销规则配置**: 定义促销条件、范围、力度、有效期
- **跨渠道规则匹配**: 自动识别用户可适用的所有促销方案
- **积分联动**: 线上下单累积积分，线下门店兑换体验服务
- **券码发放**: 一键发送定向优惠券到用户微信/短信

### 促销类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 满减券 | 满额减金额 | 满300减50 |
| 折扣券 | 指定折扣 | 秋冬装8折 |
| 积分翻倍 | 指定品类积分加倍 | 母婴类2倍积分 |
| 赠品 | 满额赠礼 | 买一送一 |
| 专属价 | 会员/导购专属价 | 内部员工价 |

### 跨渠道联动场景

**场景A**: "线上下单累积积分，线下兑换"
- 用户在天猫下单，获得积分
- 积分同步至品牌会员账户
- 用户前往门店，出示积分兑换体验服务

**场景B**: "线下试穿缺码，线上优惠券引导"
- 门店缺码，导购发送专属优惠券
- 用户通过券链接跳转小程序下单
- 订单归属关联导购业绩

## 调用示例

### 自然语言解析场景

**用户说**: "给最近30天购买过女装的会员，发送一张满500减100的冬装专享券"

**AI解析后调用**:
```
promotion_engine.create_coupon(
  campaign_name="冬装专享券",
  coupon_type="cash",
  threshold=500,
  discount=100,
  target_segment="recent_30d_female_apparel_buyers",
  channel="wechat"
)
```

---

**适用场景**:
- 营销团队策划促销活动
- 导购向顾客发放专属优惠券
- 系统自动执行生日礼遇、复购提醒
