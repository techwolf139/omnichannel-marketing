# 动态库存隔离与店仓一体化

## 技能概述

**技能名称**: inventory_ringfence

**核心能力**: 为各渠道动态圈定专属库存，支持门店作为履约中心的店仓一体化操作。

## 能力描述

### 核心功能

- **动态库存隔离(Ring-fencing)**: 大促期间为线上渠道锁定专属库存，活动结束后自动释放
- **安全库存保护**: 保护线下门店最低库存，防止被线上卖空
- **店仓一体化**: 门店具备仓库的拣货、打包、发货能力

### 隔离场景

**双11大促**:
- 平日：门店库存共享给天猫卖
- 双11当天：系统自动保护门店安全库存（每款保留3件）
- 剩余库存全部推给线上抢购
- 大促结束：未售出库存瞬间释放给全渠道

### 店仓一体化流程

```
门店收到派单 → 店员接单 → 货架找货 → 扫码核验 → 打包 → 交接骑手
```

## 调用示例

### 自然语言解析场景

**用户说**: "帮我设置双11期间女装品类门店安全库存为每款5件"

**AI解析后调用**:
```
inventory_ringfence.set_ringfence(
  category="female_apparel",
  channel="tmall",
  event="double11",
  safe_stock=5,
  time_range={start: "2024-11-11 00:00", end: "2024-11-12 00:00"}
)
```

**返回**: 隔离规则创建成功，预计影响的SKU数量和库存总量。

---

**适用场景**:
- 大促期间保护线下门店
- 设置特定渠道的专属库存
- 管理门店的履约能力配置
